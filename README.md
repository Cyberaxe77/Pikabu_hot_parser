# Пикабушный парсер "Горячего"

Десктопное приложение для получения списка горячих постов с Pikabu.ru

## Возможности

- Загрузка списка горячих постов с Pikabu
- Открытие постов в браузере по умолчанию (двойной клик или контекстное меню)
- Сворачивание в системный трей
- Обновление списка новостей по кнопке

## Требования

- Python 3.6+
- PySide6
- requests
- beautifulsoup4
- lxml
- fake-useragent

## Установка

```bash
pip install PySide6 requests beautifulsoup4 lxml fake-useragent
```

## Запуск

```bash
python main.py
```

## Использование

1. Нажмите кнопку **"Загрузить горячее"** для получения списка постов
2. Дважды кликните на пост для открытия в браузере
3. Или нажмите правой кнопкой мыши и выберите "Открыть в браузере по-умолчанию"
4. При сворачивании окна приложение уходит в системный трей
5. Клик по иконке в трее показывает/скрывает окно
